# Installation

To install the latest stable version:
```sh-session
$ npm install -g sudofy-bp
```
OR
```sh-session
$ yarn global add sudofy-bp
```

# Usage
<!-- usage -->
```sh-session
$ npx sudofy-bp react [Project-Name]
```
